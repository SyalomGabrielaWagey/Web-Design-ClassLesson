Perbedaan antara Web Aplication dan Web Server adalah, web server atau situs website hanya dapat berinteraksi satu arah. Sedangkan untuk Web Aplication atau aplikasi web, dapat melakukan interaksi, contoh dapat melakukan login. Contoh dari aplikasi web seperti gmail. Dan contoh dari situ web seperti website yang dimiliki universitas Unklab

PERBEDAAN DARI BACK END, FRONT END DAN FULL STACK DEVELOPER

Front End developer adalah profesi yang lebih berfokus pada tampilan dar suatu website. Agar lebih menarik, interaktif dan friendly. Dibutuhkan pengetahuan tentang html, css dan javascript. Contohnya adalah saat kita pertama kali membuka sebuah website, itulah bagian yang dimaksud dengan front end. Sedangkan untuk Back End developer, lebih berfokus pada bagaimana database dari website tersebut. Atau, bisa disebut juga dengan bagian belakang layar dari website tersebut. Akan bersinggungan dengan algoritma. Dan Fullstack Developer adalah bagian yang dapat melakukan keduanya, yakni front end dan back end.
